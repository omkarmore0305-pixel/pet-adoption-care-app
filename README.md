# Pet Adoption & Care App 🐾

A Flutter based mobile application that helps users adopt pets and manage pet care activities.

## Features
- User Authentication using Firebase
- Browse and adopt pets
- Add pets to favorites
- Pet care reminders
- User profile management
- Admin panel for managing pets

## Technologies Used
- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
- Firebase Storage

## How to Run
1. Clone the repository
2. Open in Android Studio / VS Code
3. Run `flutter pub get`
4. Connect emulator or Android device
5. Run the application
